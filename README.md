# Mehdi-s-portfolio

Portfolio de développeur moderne et responsive.

## 🚀 Fonctionnalités

- Design moderne et responsive
- Mode sombre/clair
- Sections : Accueil, À propos, Compétences, Projets, Contact
- Animations fluides
- Support TypeScript et Next.js

## 🛠️ Technologies

- HTML5 / CSS3
- JavaScript
- Next.js 14
- TypeScript
- React 18

## 📦 Installation

```bash
# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Build pour production
npm run build

# Lancer en production
npm start
```

## 🌐 Déploiement

Ce portfolio peut être déployé sur :
- GitHub Pages
- Vercel
- Netlify
